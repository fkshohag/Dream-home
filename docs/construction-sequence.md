# Construction Sequence — Step-by-Step

*A practical week-by-week guide. Use as a site supervision tracker.*

---

## Phase A — Pre-construction (Month 0, 2–3 weeks)

| Week | Task | Output / Deliverable |
|---|---|---|
| -3 | Soil test (auger to 10 ft, 3 points) | SBC report |
| -3 | Approval submission to Union Parishad | Receipt of application |
| -2 | Finalize structural drawings with stamped engineer | Signed drawing set |
| -2 | Tender / negotiate with 2–3 contractors | Signed BOQ contract |
| -1 | Site clearing (vegetation, levelling) | Clear plot |
| -1 | Mobilization: site shed, water connection, electricity | Site ready |
| -1 | Chalk-out: column grid pegged with chuna powder | Setting out done |
| 0 | Mobilization advance to contractor (10 %) | Build starts |

**Money out so far: ≈ BDT 5.3 lakh** (advance + soil + site setup).

---

## Phase B — Foundation & plinth (Month 1–2)

### Week 1: Excavation
- Excavate 16 footing pits, 6'-0" × 6'-0" × 5'-0" deep
- Stack good soil for plinth filling
- Inspector check: pit depth uniform, sides plumb
- **Hold point**: confirm SBC matches soil-test prediction (drop-cone test)

### Week 2: Soling + PCC + footing reinforcement
- 6" brick soling, manually packed
- 3" PCC (1:4:8) over soling
- Bind footing reinforcement on PCC (T12 @ 5" c/c BW)
- **Hold point**: engineer signs off rebar placement before pour

### Week 3: Footing concrete + column starter
- Cast all 16 footings — **continuous pour, do not break**
- Set column starter cage (T16 × 6, T10 ties @ 6") into wet footing
- Cure with wet hessian + sprinkling 14 days

### Week 4: Plinth beam + plinth filling
- Erect plinth beam shuttering, cast plinth beam (10" × 12")
- Compact plinth sand fill in 6" lifts, 95 % Proctor
- DPC layer at plinth top — 1.5" mortar 1:3 + Dr Fixit
- **Milestone**: ground level + 2'-6"

### Week 5–8: Column up to roof beam
- Erect column shuttering, tie reinforcement
- Cast columns in two lifts (up to 6' first, then to 10' second)
- Cure each lift 7 days minimum
- Verify column verticality with plumb-bob at every level

**Money out by end of Phase B: ≈ BDT 13 lakh**.

---

## Phase C — Roof slab (Month 3–4)

### Week 9: Beam shuttering + reinforcement
- Shutter main beams (B1–B4) + secondary beams (B5–B16)
- Tie beam reinforcement; ensure proper anchorage at supports
- Shutter slab — props at max 4'-0" c/c, headed boards
- Lay slab reinforcement: bottom main + dist, then chair bars, then top steel

### Week 10: Verandah cantilever + electrical conduit
- Shutter the 6'-0" verandah cantilever — extra props at edge
- Place cantilever top reinforcement (5 × T16) into beam
- Run electrical conduit on slab (2.5 sqmm + spare)
- Embed AC sleeves + solar GI sleeves at this stage

### Week 11: Slab casting — the big day
- Pre-arrange concrete (mixer + 10–12 labour) for **continuous pour**
- Order extra cement, sand, chips a day before — cannot stop mid-pour
- Slump test every batch (75–100 mm)
- Cube samples (3 per 50 cft) for lab test
- Pour from one corner, vibrate every 18", screed level
- Final smoothing within 4 hrs
- **Hold point**: Engineer sign-off on rebar before pour starts.

### Week 12–14: Slab curing (mandatory 14 days)
- Pond curing or wet hessian + sprinkler 4×/day for 14 days
- Do NOT remove props for 14 days
- Strike side shutter after 3 days, bottom slab shutter after 14
- Cube test at 7-day: should be ≥ 70 % of M20 strength

**Money out by end of Phase C: ≈ BDT 27 lakh**.

---

## Phase D — Brick & plaster (Month 5–6)

### Week 15–17: Brickwork
- 9" external walls — 1st-class brick, 1:4 mortar
- 5" internal partitions — 1st-class brick, 1:6 mortar
- Soak bricks 2 hrs before laying
- Stagger joints; rake mortar 12 mm for plaster key
- Lintels (precast or cast-in-situ) over all openings
- Door / window frames installed in shell stage (frame only, not panel)

### Week 18: External plaster
- ¾" thick cement-sand 1:4
- Build out the plinth band thickness (2'-6") — to be left exposed brick later
- Apply chajja drip groove
- Cure plaster with sprinkler 7 days

### Week 19: Internal plaster
- ½" thick cement-sand 1:6
- Cornices at ceiling-wall junction
- Run electrical chasing **after** plaster (chase with disc cutter)

### Week 20: Misc.
- Sunken floor in baths cast (4" RCC sunken + waterproof)
- Plinth band exposed-brick pointing
- Concealed plumbing 1st-fix in walls

**Money out by end of Phase D: ≈ BDT 38 lakh**.

---

## Phase E — Finishing (Month 7–8)

### Week 21: Doors + windows
- Hang door panels on frames installed in Phase D
- Install aluminum sliding window sashes
- Install MS grills on windows
- Install slat screen on W1

### Week 22: Floor tile
- Lay tile in following order: bedrooms → drawing/dining → kitchen → baths
- Tile cement-sand bed 1:4, ¾" thick (or adhesive for vitrified ≥ 16")
- Joint: 2 mm spacers, charcoal grout for living, white grout for bedrooms

### Week 23: Wall tile (kitchen + baths)
- Up to 7' in baths, full-height behind kitchen counter
- Same setting bed

### Week 24: Sanitary 2nd-fix + electrical 2nd-fix
- Mount basins, EWCs, shower mixers
- Test pressure 4 bar / 1 hr — no leaks
- Mount switches, sockets, ceiling fans, lights
- Megger insulation test ≥ 1 MΩ
- Hook up DB to service connection

### Week 25–26: Kitchen + paint priming
- Install granite counter, cabinets, sink, stove
- Apply putty + 1 primer coat throughout
- Sand & inspect; touch up

**Money out by end of Phase E: ≈ BDT 45 lakh**.

---

## Phase F — Finals (Month 9–10)

### Week 27: Final paint
- Apply 2 finish coats (emulsion inside, weather-coat outside)
- Polish segun doors (final 4th coat polyurethane)

### Week 28: Roof works
- Apply lime terracing + brick-bat coba waterproofing
- Cast roof-garden RCC planters
- Install water tank platform + tanks
- Solar GI frame (no panels yet if deferred)
- Install GI stair ladder (external) on south wall

### Week 29: Boundary + landscape
- Cast front mini-yard boundary wall
- Install main gate (W) with sliding mechanism
- Lay brick walkway from gate to verandah
- Build SW decorative screen wall
- Soft landscape: lawn, marigolds, potted palms

### Week 30: Final inspections + handover
- Engineer final inspection
- Water leak test (run all taps simultaneously)
- Electrical: check every socket, light
- Punch list of defects
- Defect remediation
- **Handover**: keys, manuals, warranties — final 8 % payment minus 2 % retention

**Money out at handover: ≈ BDT 46 lakh; 2 % retention = BDT 94,000 held for 6 months.**

---

## Phase G — Defect liability (Month 11–16)

- Move in, live in for 6 months
- Document any issues — leak, crack, electrical fault
- Release 2 % retention after engineer's final clearance

---

## Critical hold points (engineer must sign off)

1. ✅ Footing rebar BEFORE concrete pour
2. ✅ Column rebar BEFORE casting (each lift)
3. ✅ Beam + slab rebar BEFORE casting
4. ✅ Concrete cube test results (7-day & 28-day)
5. ✅ Plumbing pressure test BEFORE tile work
6. ✅ Electrical insulation test BEFORE energizing
7. ✅ Roof waterproofing BEFORE first monsoon
8. ✅ Final handover inspection

## Quality control samples (collect & preserve)

| Sample | Quantity | Test |
|---|---:|---|
| Concrete cubes (every pour) | 3 per 50 cft | 7 & 28-day compression |
| Brick crushing | 5 per 1,000 | Crushing strength |
| MS rod | 1 per 10 ton | Tensile + bend test |
| Sand silt | 1 per truck | Jar / silt test |
| Plaster sand fineness | 1 per truck | Sieve |

**Keep all test results in a binder. Demand from contractor.**

---

## Site safety

- Hard hats for all workers
- Boot-deep ditch railings during excavation
- No worker on slab during pour
- Electrical lockout-tagout when working near service
- First-aid box on site
- Local hospital phone number on site board
