# Materials Specification

*Engineering specs for every major material on the dream-home build. Use this
as the contractor's reference document.*

---

## 1. Concrete

| Grade | Use | Mix design (vol.) | Slump | Curing |
|---|---|---|---|---|
| **M20** (cube 20 N/mm²) | PCC, plinth, beams, slab | 1 cement : 1.5 sand : 3 stone chips | 75–100 mm | Pond + spray, 14 days |
| **M25** (cube 25 N/mm²) | Columns, footings | 1 : 1 : 2 | 75–100 mm | Pond + spray, 14 days |
| PCC 1:4:8 | Below footings | 1 : 4 : 8 (cement : sand : brick chips) | 50 mm | 3-day sprinkler |

**Cement**: OPC 53-grade for structural; PPC 43-grade acceptable for plaster.
**Water-cement ratio**: max 0.45 for M25; max 0.50 for M20.
**Site test**: take 3 cube samples per pour, test at 7 / 14 / 28 days.

## 2. Reinforcement (steel)

| Type | Grade | Use |
|---|---|---|
| Main bars (T12 / T16) | **Fe-500W (HYSD)** | Footings, columns, beams, slab main |
| Distribution bars (T8 / T10) | Fe-500 | Slab distribution, ties, stirrups |
| Tying wire | 18 SWG annealed | All tie operations |

**Lap length**: 50 × diameter for tension, 40 × diameter for compression.
**Cover**: 50 mm for footings, 40 mm for columns, 25 mm for beams, 20 mm for slabs.
**Always provide mill test certificate (MTC)**. Reject rusty rod with > 0.5 mm pitting.

---

## 3. Brick

| Spec | Value |
|---|---|
| Size | 9.5 × 4.5 × 2.75 inches (standard BD modular) |
| Class | 1st class (M-class) only for external walls; 2nd class acceptable for internal partitions if budget-tight |
| Crushing strength | ≥ 1,200 psi |
| Water absorption | ≤ 15 % by weight |
| Efflorescence | Nil to slight |
| Source | Local brickfield with kiln-fired brick (avoid sun-dried) |

**Mortar**: 1 cement : 4 sand for external 9" walls; 1 : 6 for internal 5" partitions.
**Soak bricks 2 hours before laying** to prevent water absorption from mortar.

## 7. Doors

### Main door (D1)
- **Material**: solid segun (teak) flush door, 4" thick (with internal block-board core)
- **Size**: 4'-0" × 7'-6"
- **Finish**: polyurethane natural-tone, 4 coats
- **Hardware**: brass mortise lock + brass handle on rosette; 3 × 4" SS hinges; brass door viewer (peephole); brass kick plate (optional)
- **Frame**: segun, 4" × 3", anchored with steel plates @ 18" c/c

### Interior segun doors (D2–D9)
- **Material**: solid segun ply 1¾" thick + 1¼" segun frame
- **Size**: 3'-0" × 7'-0" (or as noted)
- **Finish**: polyurethane natural-tone, 3 coats
- **Hardware**: brass mortise lock (standard) + door knob; 3 × SS hinges

### PVC bath doors (D7, D10)
- **Material**: PVC flush door with PVC frame
- **Size**: 2'-6" × 7'-0"
- **Finish**: factory white
- **Hardware**: PVC + SS hinges; bolt-style latch

## 8. Windows

### Aluminum sliding windows
- **Frame**: anodized aluminum, charcoal-grey, 60 × 40 mm sections
- **Glass**: 5 mm tinted toughened (for safety on large panes ≥ 1.5 m²)
- **Track**: double-track aluminum with PE rollers
- **Mesh**: PVC-coated stainless steel mesh inside (mosquitos)
- **Grill**: 1" × ¼" MS flat bar, vertical pattern @ 4" c/c, matte-black powder-coat
- **Sealant**: silicone, weather-grade

### Bath ventilators (W5, W9)
- **Frame**: PVC or aluminum, 2'-0" × 2'-0"
- **Glass**: frosted 4 mm
- **Grill**: 1" × ¼" MS flat, simple pattern

### Slat screen (W1 — drawing room west)
- **Slats**: 1" × 3" segun (teak), polyurethane finish
- **Spacing**: 4" c/c (centre-to-centre)
- **Height**: 8'-0" tall
- **Frame**: 2" × 3" segun top + bottom rails
- **Anchoring**: into RCC slab above (chajja) + plinth band below

## 15. Site quality control

| Check | Frequency | Action if fail |
|---|---|---|
| Cube test (3 nos per pour) | Every concrete pour | Re-pour if 28-day strength < 80 % of spec |
| Slump test | Every truck/batch | Reject batch if slump > 125 mm or < 50 mm |
| Steel mill cert | Every steel delivery | Reject batch |
| Brick crushing | Random 5 from every 1,000 | Reject lot if > 20 % fail |
| Plumbing pressure test | Before tile work | Hold 4 bar for 1 hour |
| Electrical insulation test | Before energizing | Megger test ≥ 1 MΩ |
