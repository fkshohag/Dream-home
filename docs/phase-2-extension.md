## 3. What's already built-in (you don't pay for these again)

- ✅ **All 16 columns** — sized for 3 floors total (12"×12" M25 with 6-T16)
- ✅ **Foundation** — sized for 3-floor load
- ✅ **Plinth beam** + ground-floor slab + beams — done
- ✅ **Stair opening in the slab** — already cast as a removable plug; just unplug it
- ✅ **Column starter bars** — 4'-0" projection at every column, oiled & sand-packed; just splice and continue up
- ✅ **Plumbing risers from the bath stack** — sealed with brass plug at roof level; just connect new bathrooms
- ✅ **AC sleeves & electrical riser** — already cast in walls/slab; just pull new cables
- ✅ **Solar conduit** — already runs from roof to ground; will need extension up

---

## 4. What you have to add

### Structural
1. **Splice new column rebar** onto the existing 4'-0" starters; cast 16 new columns up to 10'-6" above ground roof (i.e., to new 1st-floor slab level).
2. **New ring of beams** at first-floor slab level (same sizes as ground roof: 10"×12" main beams).
3. **New first-floor slab** — 5" two-way RCC, M20, same reinforcement scheme.
4. **New balcony cantilever** above verandah — beams continue from B17 line; new T16 top steel.
5. **Mumty walls** + small roof slab (8'-0" × 10'-0") above the stair opening.
6. **New parapet** on the new roof — 3'-6" tall.

### MEP (Mechanical, Electrical, Plumbing)
1. **Lift water tanks** from current roof to new top roof; extend pipe risers.
2. **Re-mount solar panels** on new top roof (or replace if 8+ years old).
3. **Extend electrical conduit + cable** from ground DB to new 1st-floor sub-DB.
4. **New plumbing branches** for master bath + common bath; tie into existing wet stack.

### Finishing
1. Brickwork — 9" external + 5" partitions.
2. Plaster + paint.
3. Floor tiles + wall tiles.
4. Doors + windows.
5. New balcony railing + chajja.

## 7. Phase-2 prep work to do NOW (during Phase-1 build)

These cost almost nothing now but make Phase-2 vastly easier:

| Item | Cost today | Phase-2 benefit |
|---|---:|---|
| Starter bars cast monolithic + protected | included | Saves BDT 2 lakh in splice work |
| Removable stair plug instead of solid slab | + 12,000 | Saves BDT 80,000 in slab cutting + re-pour |
| Embedded vertical PVC conduit in 4 columns (water rise, electrical rise) | + 8,000 | Saves BDT 25,000 in chasing walls later |
| Stub plumbing risers at roof level (capped) | + 10,000 | Saves BDT 30,000 in re-tapping vertical stack |
| Engineer's stamped Phase-2 drawing kept in safe | + 20,000 | Saves BDT 60,000 in re-design |
| Photograph all rebar before pour | – (free!) | Lets future engineer verify what's actually there |

**Total Phase-2 prep: BDT 50,000 today saves ≈ BDT 4 lakh later.** Already included in Phase-1 BOQ.
