# বাড়ি হস্তান্তর চেকলিস্ট (Handover & Commissioning Checklist)

> ঠিকাদার কাজ শেষে এই full checklist যাচাই করতে হবে। যেকোনো item-এ ❌ থাকলে handover স্থগিত।

---

## ১. গঠনগত (Structural)

### ভিত্তি ও কলাম
- [ ] কোনো visible crack নেই (column, beam, slab)
- [ ] Plinth wall straight + level
- [ ] কলাম plumb (laser level দিয়ে check)
- [ ] DPC course intact
- [ ] **Phase-2 starter rebar (ছাদের উপরে ৪'-০") protected** — anti-rust coating applied

### ছাদ
- [ ] Slab thickness 5" (test core drilling if required)
- [ ] কোনো leak নাই (পানি জমিয়ে 24 hour test)
- [ ] Parapet wall ৩'-৬" উঁচু, plumb, anchored
- [ ] Roof drain slope 1:80 functioning
- [ ] Future stair opening (8'×5') precast plug fitted + sealed
- [ ] Cantilever verandah deflection minimum (laser check)

### দেয়াল
- [ ] কোনো hollow plaster নাই (tap test)
- [ ] কোনো settlement crack নাই
- [ ] Internal walls plumb
- [ ] External walls smooth surface

## ২. দরজা ও জানালা (Doors & Windows)

### সব দরজা চেক:
- [ ] D1 (মূল ৪×৭.৬'): smooth open/close, digital lock + key working
- [ ] D2 (সার্ভিস): MS grill anchored, lock smooth
- [ ] D3-D9 (ভিতরের): all hinges silent, locks working
- [ ] D7, D10 (PVC bath): privacy lock cylinder
- [ ] D11 (wardrobe sliding ৩টা): track smooth, soft-close

### সব জানালা চেক:
- [ ] W1 (drawing room ৮'×৭'): 3-panel sliding smooth, MS grill anchored
- [ ] W2-W11 (sliding): rollers smooth, locks engage
- [ ] W5, W9 (bathroom frosted): exhaust fan working
- [ ] W12 (stair vent): fixed, sealed
- [ ] All mosquito mesh installed and intact
- [ ] All glass clean and intact (no cracks)

---

## ৩. প্লাম্বিং (Plumbing)

### পানি সাপ্লাই
- [ ] Tube well pump চালু হয় (1 minute test)
- [ ] Roof tank ভরে — কত সময়ে full check
- [ ] সব ট্যাপ থেকে পর্যাপ্ত pressure
- [ ] **Pressure test:** ০.৩ MPa, ৩০ মিনিট — no leak
- [ ] Geyser ২টা চালু হয় + warm water আসে (10 min test)
- [ ] Drinking water tap (RO/purifier filter installed)

### পয়ঃনিষ্কাশন
- [ ] সব sink/basin smooth drainage
- [ ] WC flush 1 bucket water, no leak
- [ ] Shower drainage no pooling
- [ ] Smoke test (drainage system) — no leak
- [ ] Vent pipe outlet roof-এ visible, mosquito mesh

### সেপ্টিক
- [ ] সেপ্টিক ট্যাংক empty এবং vent functioning
- [ ] Soak pit drainage tested
- [ ] No bad smell

### Fixtures
- [ ] WC 2টা — flush, no leak
- [ ] Basin 2টা — drain functional
- [ ] Shower mixer 2টা — hot+cold mixing OK
- [ ] Kitchen sink — drain + tap functional
- [ ] All shut-off valves operational

---

## ৪. বিদ্যুৎ (Electrical)

### মেইন
- [ ] মিটার installed + sealed by REB/DPDC
- [ ] DB তে label-এ proper circuit naming
- [ ] Main MCB 63A + ELCB 30mA tested

### Per circuit
- [ ] Lighting circuit: all lights working
- [ ] General socket circuit: all sockets tested with multimeter
- [ ] Kitchen circuit: high-load appliances (fridge, microwave)
- [ ] AC circuit: 1.5 ton load test (15-min run)
- [ ] Geyser circuit: heating test (15-min run)
- [ ] Pump circuit: motor + dry-run protection

### Earthing
- [ ] Earth resistance < ৫ Ω (multimeter)
- [ ] All sockets have earth pin functional
- [ ] All metal items connected (water tank, fridge, AC)

### Tests
- [ ] **Megger insulation test:** ≥ ১ MΩ all circuits
- [ ] **ELCB trip test:** push button activates within 30 ms
- [ ] **MCB trip test:** overload disconnects properly
- [ ] All wiring concealed; no exposed joints
- [ ] Voltage at sockets: 220V ± 5%

### Solar
- [ ] 6 panels installed and connected
- [ ] Inverter producing power (daytime test)
- [ ] Battery (if any) charging properly
- [ ] Net metering connected with REB

---

## ৫. ফিনিশিং (Finishing)

### টাইলস
- [ ] কোনো hollow tile নাই (tap test entire floor)
- [ ] Grout intact, no crack
- [ ] Skirting tile flush
- [ ] No staining
- [ ] Bathroom waterproofing tested (pour bucket water, no seepage 24h)

### রং (Paint)
- [ ] External wall full coverage, no patches
- [ ] Internal wall full coverage
- [ ] No drip marks
- [ ] All edges crisp (taping done)
- [ ] Door/window frame painted (separate)
- [ ] Ceiling smooth, no roller marks

### Wood work
- [ ] Door polish smooth, no scratches
- [ ] Wardrobe doors aligned (no gap > 2mm)
- [ ] All drawers smooth open/close
- [ ] All hinges silent
- [ ] No nail holes visible

### রান্নাঘর
- [ ] Granite counter — no chips, joint epoxy clean
- [ ] All cabinet doors aligned + soft-close
- [ ] Drawer slides smooth (full-extension)
- [ ] Sink installed properly with silicone seal
- [ ] Gas line tested (no leak)
- [ ] Chimney suction tested
- [ ] Microwave shelf level

## ৯. Final inspection meeting

### Attendees:
- [ ] মালিক + পরিবার
- [ ] ঠিকাদার / প্রধান মিস্ত্রি
- [ ] ইঞ্জিনিয়ার
- [ ] (Optional) Architect

### Inspection sequence:
1. ✅ বাইরে থেকে শুরু — boundary wall + gate
2. ✅ ভেরান্ডা + main entry
3. ✅ Drawing room → dining → kitchen
4. ✅ Bedroom 1 → att. bath
5. ✅ Bedroom 2 + 3 → common bath
6. ✅ Stair space
7. ✅ Roof (if accessible) — water tank, solar, garden
8. ✅ ভিত্তি (অভিযোগ — visible signs)
9. ✅ Septic + tube well + landscape
10. ✅ Final document handover
