# বার্ষিক রক্ষণাবেক্ষণ সূচি (Yearly Maintenance Schedule)

> এই কাগজ পরিবারের জন্য — বাড়ি handover হওয়ার পর ১ বছর থেকে শুরু।

---

## মাসিক চেক (Monthly — প্রতি মাসে ১ বার)

### সিভিল
- [ ] দেয়ালে কোনো ফাটল আছে কিনা চেক
- [ ] দরজা/জানালা smooth operate করে কিনা
- [ ] ছাদে পানি জমা আছে কিনা (বর্ষাকালে সাপ্তাহিক)

### প্লাম্বিং
- [ ] কোনো leak আছে কিনা — sink, taps, WC
- [ ] পানির pressure ঠিক আছে কিনা
- [ ] সব valve operate করে কিনা
- [ ] Hot water (geyser) চালু হয় কিনা

### বিদ্যুৎ
- [ ] সব socket, switch কাজ করে কিনা
- [ ] MCB / ELCB test (push button)
- [ ] কোনো burning smell নেই
- [ ] Surge protector intact

### রান্নাঘর
- [ ] Gas leak (সাবান পানি দিয়ে)
- [ ] Chimney exhaust ভালো কাজ করে
- [ ] Fire extinguisher pressure gauge — green zone
- [ ] Smoke alarm beep test

### বাগান
- [ ] গাছে পানি দেওয়া (গ্রীষ্মে weekly, বর্ষায় কম)
- [ ] দুর্বল গাছে সার
- [ ] Pest check (insect, fungus)
- [ ] Lawn mowing (১৫ দিন interval)

---

## ৩-মাসিক চেক (Quarterly)

### সিভিল
- [ ] ছাদের waterproofing স্তর intact কিনা
- [ ] Parapet wall crack
- [ ] Floor tile loose কিনা (tap test)
- [ ] Wall painting intact

### প্লাম্বিং
- [ ] পানির ট্যাংক clean
- [ ] Tube well pump suction pressure
- [ ] Drainage clearance (slow flow → block?)
- [ ] Septic tank vent pipe mosquito mesh

### বিদ্যুৎ
- [ ] Earth wire continuity test (multimeter)
- [ ] Inverter / UPS battery test
- [ ] CCTV recording quality
- [ ] All bulbs working

### Hardware
- [ ] Door hinges oil
- [ ] Window slider track clean + lubricate
- [ ] Lock cylinder smooth operation
- [ ] Furniture screw tightening

### বাগান
- [ ] গাছের pruning (fruit trees)
- [ ] Lawn fertilizer (NPK)
- [ ] Mulching (পাতার আস্তরণ)
- [ ] Compost pit turn over

---

## ৬-মাসিক চেক (Half-yearly)

### সিভিল
- [ ] পুরো বাড়ির visible damage assessment
- [ ] Boundary wall crack / coping intact
- [ ] Gate operation + lock service

### প্লাম্বিং
- [ ] **পানির ট্যাংক পরিষ্কার** (প্রতি ৬ মাসে - sediment, mosquito)
- [ ] Geyser thermostat test + descale
- [ ] Septic tank IC inspect
- [ ] Rainwater filter cleanup

### বিদ্যুৎ
- [ ] Megger insulation test (electrician)
- [ ] Earth resistance test
- [ ] DB cleanup + dust removal
- [ ] All AC servicing (filter, gas)

### Roof
- [ ] Solar panel cleaning (dust + bird droppings)
- [ ] Solar wiring connection check
- [ ] Water tank lid + mesh
- [ ] Roof drain clearance
- [ ] Future stair opening waterproof check

### Termite / pest
- [ ] Wood treatment check (doors, wardrobe)
- [ ] Termite tunnels visible?
- [ ] Pest control spray (rooms + corners)

---

## বার্ষিক চেক (Yearly)

### Structural
- [ ] **ইঞ্জিনিয়ারের visit** — সব structural element inspect
- [ ] কলামে hairline crack
- [ ] বিম-স্ল্যাব এর underside crack
- [ ] Foundation settlement check (level)
- [ ] Plinth + DPC integrity
- [ ] Phase-2 starter rebar inspect (যদি rust থাকে — anti-rust coat)

### Roof major
- [ ] Brick-bat coba layer check
- [ ] Waterproofing membrane integrity
- [ ] Parapet weep holes clean
- [ ] Cantilever verandah deflection check (laser level)

### MEP major
- [ ] **পানি quality test** — Arsenic, iron, bacteria
- [ ] Septic tank — পরিপূর্ণ হওয়া ৬০-৮০% হলে empty করতে হবে
- [ ] Inverter battery health
- [ ] Solar production check vs nameplate

### Finishing
- [ ] Repainting decision — exterior paint life ৩-৫ years, interior ৫-৭ years
- [ ] Wood polish (door, furniture) + termite treatment
- [ ] Granite re-polish (যদি অনেক scratch)
- [ ] Window glass replacement (যদি ভাঙা/চিড়)

### Documents
- [ ] Insurance renewal (house + valuables)
- [ ] Property tax payment receipt
- [ ] Utility connections (electric, gas, water) bill paid
- [ ] Emergency contact list update
- [ ] All warranty cards organize

---

## ৩-বছরিক চেক (Every 3 years)

- [ ] **Septic tank pump out** (full cleaning)
- [ ] Underground rainwater tank deep clean
- [ ] Exterior paint (full repainting)
- [ ] Boundary wall + gate full repainting
- [ ] Solar panel deep clean (technical service)
- [ ] All locks: cylinder change (security)
- [ ] CCTV system upgrade check

---

## ৫-বছরিক চেক (Every 5 years)

- [ ] **Interior repainting** (full)
- [ ] Tile re-grouting (special epoxy needed?)
- [ ] Door hinges + hardware replacement (high-use)
- [ ] All fittings (faucets, showers) replacement
- [ ] Electrical wiring insulation test (megger ৫ MΩ minimum)
- [ ] AC compressor service / replacement consideration
- [ ] Solar inverter capacitor check

---

## ১০-বছরিক চেক (Every 10 years)

- [ ] **Structural engineer audit** — সব structural element
- [ ] Foundation settlement check (precision)
- [ ] Roof complete waterproofing replacement (consider)
- [ ] Septic tank wall + bottom integrity
- [ ] Tube well casing inspection
- [ ] Major electrical upgrade (technology change)
- [ ] Solar panel performance evaluation (replacement?)
- [ ] **Phase-2 second floor implementation feasibility study**

## পরিচর্যা টিপস

### ✅ যা করবেন
- প্রতি মাসে ১ ঘন্টা বাড়ির inspection
- Photo নিয়ে before/after comparison
- Service vendor-এর contact list maintain
- Receipts রাখুন (warranty + tax)
- Maintenance log book/diary

### ❌ যা করবেন না
- Cheap labor দিয়ে important repair (cost long-term বেশি)
- Annual inspection skip
- Small crack ignore (বড় হয়ে গেলে major repair)
- DIY electrical (qualified electrician only)
- Untreated leak (waterproofing damage)
