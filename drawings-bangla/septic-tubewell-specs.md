### ঘ. কাজের ক্রম

1. মাটি কাটা: ৭'×৫' উপরে, ৬' গভীর
2. নিচে ৬" ইট সোলিং + ৩" PCC
3. Bottom RCC slab ঢালাই + ৭ দিন কিউরিং
4. ১০" দেয়াল গাঁথা ১:৪ মর্টারে
5. Baffle wall গাঁথা
6. ভিতরে waterproof plaster
7. Inlet, outlet, vent pipe বসানো
8. Top slab ঢালাই + manhole opening
9. মাটি ভরাট + প্লেট কম্প্যাকশন
10. ⚠️ প্রথমবার ব্যবহারের আগে ৩-৪ দিন পানি ভরে test

## ৩. ডিপ নলকূপ (Deep Tube Well)

### ক. বেসিক স্পেসিফিকেশন

| বিষয় | স্পেসিফিকেশন |
|---|---|
| Diameter | ৪" (casing pipe) |
| গভীরতা | ৩০০' - ৬০০' (এলাকা অনুসারে — পল্লী জল সরবরাহ অফিসে check) |
| Casing pipe | ৪" PVC heavy duty / GI |
| Strainer pipe | ২০-৩০ ফুট (aquifer level এ) |
| Submersible pump | ১ HP, ১০০' গভীরে |
| Delivery pipe | ১" UPVC schedule 40 |
| Control panel | Capacitor + on/off + dry-run protection |

### খ. drilling/বরিং প্রক্রিয়া

1. **এলাকার পানি depth জেনে নিতে হবে** (BWDB website বা পল্লী জল সরবরাহ অফিসে)
2. Site selection: 
   - বাড়ির ২০' এর মধ্যে (electrical run কম)
   - সেপ্টিক / soak pit থেকে **ন্যূনতম ৩০' দূরে**
   - পুকুর / নালা থেকে দূরে
3. Rotary drilling rig দিয়ে drill (গ্রামের local drill team)
4. প্রতি 20' depth এ soil sample রাখতে হবে
5. Aquifer hit করলে gravel pack + strainer pipe insert
6. Casing pipe seal (cement grouting top 50')

### গ. Pump installation

- Submersible pump 100' গভীরে hang করতে হবে (rope + steel cable)
- Control panel মাটির 2' উপরে, lockable enclosure
- Cable: 3-core flat submersible cable
- Earth: pump body থেকে আলাদা earth wire

### ঘ. Concrete platform

- নলকূপের চারপাশে ৩'×৩' কংক্রিট platform
- ৩" পুরু RCC, M১৫
- ১:১০০ ঢাল outward (পানি ছিটিয়ে গেলে যাতে নলকূপ মাঝখান পরিষ্কার থাকে)
- পাশে drainage channel ৬" চওড়া

### ঙ. পানির গুণমান পরীক্ষা (Water Quality Test)

**শুরুতে এবং প্রতি বছর check করতে হবে:**

| Parameter | নিরাপদ মাত্রা |
|---|---|
| Arsenic | < ০.০৫ mg/L (WHO standard < ০.০১) |
| Iron | < ১ mg/L |
| Manganese | < ০.১ mg/L |
| pH | ৬.৫ - ৮.৫ |
| TDS | < ১০০০ mg/L |
| Total coliform | কখনোই থাকা যাবে না |

**Test কোথায় করাবেন:**
- উপজেলা পল্লী জল সরবরাহ অফিস (DPHE)
- ICDDR,B ল্যাব
- স্থানীয় কলেজ / বিশ্ববিদ্যালয় ল্যাব

⚠️ **Bangladesh এ অনেক এলাকায় arsenic problem আছে** — শুরুতেই test অবশ্যই করবেন

### চ. যদি arsenic থাকে

- ৫০০ ফুটের নিচের aquifer থেকে বরিং (deeper layers usually safer)
- যদি deeper-ও contaminated, household-এ arsenic filter বসাতে হবে
- SONO filter / SIDKO filter / pillared sand filter
