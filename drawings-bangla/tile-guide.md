## ৩. কেনার আগে যা চেক করবেন

### ক. টাইলের গুণমান চেক

✅ **Surface inspection:**
- কোনো crack নেই
- Glaze সম্পূর্ণ uniform
- Color match প্রতি tile-এ (একই batch)
- Edge straight (rectangular check)
- Size variation: ±০.৫mm এর বেশি না (PEI rating দেখুন)

✅ **Sound test:** ২টা tile একসাথে tap করলে clear ringing sound হবে (cracked tile dull sound দেবে)

✅ **Water absorption test:**
- Vitrified: <০.৫% (premium)
- Ceramic: <৩%
- বাথরুমের জন্য vitrified or ceramic gloss

### খ. PEI Rating (durability)

| PEI Class | কোথায় ব্যবহার |
|---|---|
| Class 1 | শুধু দেয়াল (no foot traffic) |
| Class 2 | বাথরুম, শোবার ঘর |
| Class 3 | বসার ঘর, dining (light traffic) |
| Class 4 | বসার ঘর + heavy traffic এলাকা |
| Class 5 | Commercial / public (industrial) |

আপনার বাড়ির জন্য: PEI 3-4 যথেষ্ট

### গ. Anti-skid R rating

| R rating | কোথায় |
|---|---|
| R9 | সাধারণ অভ্যন্তরীণ |
| R10 | রান্নাঘর, ভেরান্ডা |
| R11 | বাথরুম, ভেজা area |
| R12+ | Pool, industrial wet |

বাথরুম + ভেরান্ডায়: minimum R11

### ঘ. Batch number

⚠️ **একই room-এ একই batch number এর টাইল**
- ভিন্ন batch = ভিন্ন color shade
- কেনার আগে invoice-এ batch number লিখিয়ে নিন
- ১০% বেশি কিনে রাখুন একই batch থেকে — future repair এর জন্য

## ৪. বসানোর নিয়ম (Installation Rules)

### ক. কাজের ক্রম

1. **Surface preparation:**
   - মেঝে level হতে হবে (sloping দরকার হলে সেটা plan অনুযায়ী)
   - Wall plaster cured (২৮ দিন পর)
   - সব আবর্জনা/dust পরিষ্কার

2. **Marking & layout:**
   - Room-এর center mark করুন
   - সেখান থেকে চার দিকে measure
   - Edge-এ যে cut tile আসবে সেটা যেন ছোট না হয় (minimum half-tile)
   - Door-এর threshold থেকে প্রথম পুরো tile

3. **Adhesive mixing:**
   - **Cement-based adhesive (BondLok / Roff / Mapei):**
     - পানি + powder mix, ৫-১০ মিনিট rest
     - মেঝেতে notched trowel দিয়ে spread
   - **পুরানো method - cement mortar:**
     - ১:৩ সিমেন্ট:বালি
     - পুরুত্ব ১"-১.৫"

4. **Tile placement:**
   - ২mm spacer between tiles
   - প্রতি tile পিছন থেকে slight back-buttering (extra adhesive)
   - Tap with rubber mallet — flat level check
   - Tile-এর underside কোনো hollow নেই (tap test)

5. **Joint filling (grouting):**
   - ২৪-৪৮ ঘন্টা পর tile setting হলে
   - **Cement grout (cheaper):** white/grey cement + acrylic latex
   - **Epoxy grout (premium):** Roff Rakshak — waterproof, stain proof
   - ⚠️ Bathroom + kitchen-এ epoxy grout recommended

6. **Cleaning:**
   - Grout setting হওয়ার ৩০-৬০ মিনিট পর diagonal wipe
   - Cement haze cleaner (tile haze remover) দিয়ে শেষ পরিষ্কার

### খ. প্রতি room-এ starting point

| Room | কোথা থেকে শুরু |
|---|---|
| বসার ঘর | Center থেকে (large room) — তবে dominant view থেকে দেখলে যেন centred দেখায় |
| শোবার ঘর | Center থেকে |
| রান্নাঘর | কাউন্টারের ১ম পুরো tile (corner থেকে) |
| বাথরুম | Door threshold থেকে |
| ভেরান্ডা | বাড়ির দিক থেকে (yard side cut) |

⚠️ **গুরুত্বপূর্ণ:** যেদিকে cut tile আসবে সেটা door-এর opposite side বা যেখানে কম দেখা যায়।

### গ. বাথরুম দেয়ালে tile বসানো

1. মেঝে tile আগে — wall tile পরে
2. দেয়ালের নিচ থেকে শুরু — first row পরিষ্কার, পুরো tile
3. উপরের দিকে যেতে যেতে cut tile আসবে (ceiling line-এ)
4. শাওয়ারের জন্য niche (১৮"×১২"×৪" deep) আগে frame করতে হবে
5. Towel rail anchor point আগে mark
6. Mirror behind ply backing আগে
7. সব accessories (soap dish, etc.) plumbing শেষ হওয়ার পর

### ঘ. বাথরুমের ঢাল (Slope)

- শাওয়ার drain এর দিকে ১:৫০ slope
- বাথরুম পুরো ১:১০০ slope (door এর opposite direction)
- 🚿 শাওয়ার area separate করতে glass partition (optional)

## ৫. বাথরুম waterproofing (টাইলের আগে)

⚠️ **টাইল বসানোর আগে অবশ্যই waterproofing করতে হবে।**

1. মেঝে: cement-based waterproof আস্তরণ ২ কোট (Dr.Fixit / Sika)
2. দেয়াল: শাওয়ার এর চারপাশে ৪' উচ্চতা পর্যন্ত
3. পুরো বাথরুমের চারপাশে নিচ থেকে ১ ফুট পর্যন্ত
4. Drain pipe joint-এ extra coat
5. Application: brush বা trowel দিয়ে
6. ২৪ ঘন্টা dry, তারপর ২য় coat
7. **Pressure test:** ৪৮ ঘন্টা পানি জমা রেখে test (drain block করে)

## ৬. রান্নাঘরের special considerations

### Backsplash
- Counter থেকে wall cabinet পর্যন্ত (১৮" tall) **MUST**
- Stove এর পিছনে full-height (ceiling পর্যন্ত) recommended — oil splatter
- Subway tile (৩"×১২") সবচেয়ে popular
- Color: light cream/white (clean look + light reflection)

### Floor
- Anti-skid R10+
- Easy-to-clean smooth surface
- Avoid: glossy floor tile (slippery + oil show)

### Counter material (granite)
- ⚠️ Granite under tile separately install
- Granite-counter আগে install (carpenter কাজ শেষ হলে)
- তারপর backsplash tile

## ১০. ভুল করবেন না (Common Mistakes to Avoid)

❌ **মেঝে level না করে tile বসানো** — water log হবে, tile crack হবে  
❌ **Spacer ছাড়া tile বসানো** — uniform joint দেখাবে না  
❌ **Cement mortar ১.৫" এর বেশি পুরু** — tile pop-up হবে  
❌ **Wet tile-এর উপরে walk** — settling disturb হবে  
❌ **Same room-এ ভিন্ন batch tile** — color difference দৃশ্যমান  
❌ **Bathroom-এ glossy floor tile** — slip + injury risk  
❌ **Sealant ছাড়া joint** — মাটি / আবর্জনা ঢুকে black হবে  
❌ **প্লাস্টার dry হওয়ার আগে tile** — adhesion fail
