## 2. Items detail

### A. Overhead water tank

| | Value |
|---|---|
| Capacity | 2 × 1,000 L = 2,000 L total (Gazi or RFL black HDPE tank, food-grade) |
| Location | NE corner of roof, above the wet-stack column (C4 area) — shortest pipe runs |
| Platform | 3'-0" tall RCC plinth, 8'-0" × 8'-0", on 4 columns C3-C4-C7-C8 |
| Inlet | 1" PVC from submersible pump, with float valve in master tank |
| Outlet | 1" PVC main to vertical bath stack; ½" branch to kitchen sink |
| Overflow | 1" PVC pipe discharging to nearest downspout |
| Cleanout | Yearly — drain via bottom 1" gate valve |

### B. Solar panel array

| | Value |
|---|---|
| Rated capacity | 3 kW (6 × 400 W bifacial mono-crystalline, e.g. JA Solar or Trina) |
| Frame | Galvanized iron, 15° tilt facing **WEST** (peak generation in late afternoon) |
| Mount | Bolted to embedded GI sleeves cast in roof slab (4 sleeves per panel = 24 total) |
| Footprint | ~ 16'-0" × 10'-0" |
| Inverter | 5 kW grid-tied hybrid (battery-less initially; battery-ready terminals); located inside R10 |
| Cable route | DC cables through 1" PVC conduit cast in roof slab → wall chase → inverter |
| Net-metering | Apply to local REB after commissioning |
| Expected annual yield | ~ 4,000 kWh; meets ~70 % of family annual consumption |

### C. Roof garden zone

| | Value |
|---|---|
| Footprint | 12'-0" × 12'-0" in SE quadrant (over BR3) |
| Construction | 6"-tall RCC kerb walls, 6" HDPE sheet liner, 4"-thick gravel drainage, 6"-thick soil mix |
| Drainage | One 2" PVC drain through slab, sleeved & waterproofed, discharging to NE downspout |
| Suggested plants | Tulsi, chilli, mint, lemon (potted), curry leaf, marigolds |
| Maintenance access | 18"-wide paved path on three sides |
| Load on slab | Saturated soil ~ 110 psf; slab can carry — designed for 100 psf live |

### D. Future stair opening (Phase-2)

| | Value |
|---|---|
| Opening size | 8'-0" × 5'-0" — large enough for 6'-6" stair flight + landing |
| Location | Directly above ground-floor stair space (R10) |
| Edge framing | 4 × T16 around perimeter of opening, cast monolithic with slab |
| Plug | 6"-thick precast RCC slab plate, sized 8'-0" × 5'-0" minus 1" all around for grouting; sits on a 3" continuous concrete ledge inside the opening |
| Sealing | EPDM gasket strip + sand-cement grouting around perimeter; waterproofing membrane lapped over the joint |
| Removal | When Phase-2 begins: crowbar lift the plug, slide off, store for re-use as workshop slab |

### E. Roof finish & waterproofing

| Layer (top to bottom) | Thickness | Material |
|---|---:|---|
| Surface tile (open terrace + planter path) | ½" | Anti-skid terracotta tile |
| Brick-bat coba | 2" | Crushed brick + lime mortar, sloped 1:80 |
| Lime terracing | 1" | Slaked lime + surkhi (chemical waterproof) |
| Cement-sand layer | ½" | 1:3 |
| RCC slab | 5" | M20 |
| **Total roof thickness** | **9"** | |

Optional upgrade: **chemical waterproofing membrane** (Saver / Dr Fixit Roofseal) — adds ~ BDT 15,000 but extends life 20+ years vs. 12.

### F. Parapet

- 3'-6" tall, 5" thick brick wall, plastered, terracotta-tile capped
- Internal face: continuous 4" wide concrete coping for tank pipe runs
- 4"-square PVC weep holes every 6'-0" at slab level

---

## 3. Roof access

Currently, the only way up to the roof is via an **external GI ladder** mounted on
the south wall (next to the south back door D2). Spec: 12'-0" tall, 16" wide,
caged from 6'-0" up.

When Phase-2 first floor is built, this external ladder will be removed and
replaced by the internal staircase opening up to a mumty (stair cover).

## 4. Maintenance schedule

| Task | Frequency |
|---|---|
| Inspect waterproofing membrane | Annually (before monsoon) |
| Clean downspouts + weep holes | Twice yearly (Apr + Oct) |
| Clean & flush water tanks | Yearly |
| Clean solar panels | Monthly during dry season, after every dust storm |
| Re-cover roof-garden planter beds with mulch | Yearly |
| Inspect parapet for cracks | Annually |
| Tighten solar frame bolts | Annually |
| Check column starter-bar caps (oil + sand) | Annually until Phase-2 |
