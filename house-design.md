# House Design — Master Index (48' × 25', 1,200 sqft)

> Single-story modern tropical home for a Bangladeshi village plot, engineered for a future 2nd floor. Layout based on `diagram/design 2.png`. This file indexes every design document and drawing.

---

## 1. Project Snapshot

| Item | Value |
|---|---|
| Footprint | 48'-0" (N–S) × 25'-0" (E–W) = **1,200 sqft** |
| Verandah | 25' × 6' projecting south (+150 sqft) |
| Bedrooms | 3 (2 masters w/ attached bath + 1 with common bath) |
| Other rooms | Drawing, Dining, Kitchen, 3 baths, Stair, Verandah |
| Main entry | SOUTH (verandah + main gate) |
| Secondary entry | WEST door (into Drawing room) |
| Structure | RCC frame, 15 columns (3×5), isolated footings |
| Future | Designed for G+1 (2nd floor) |
| Core cost | **≈ 50–52 lakh** (~4,280 BDT/sqft, mid tier) |
| Grand total (core + extended) | **≈ 84 lakh** |

---

## 2. Key Design Decisions

1. **Central-dining pinwheel** — dining is the hub; 4 corners hold Drawing (SW), Room 1 (SE), Room 2 master (NE), Room 3 master (NW).
2. **Service spine S→N:** Stair (at south entry) → Dining (center) → Kitchen (on north wall).
3. **Kitchen on the north external wall** with window + exhaust → cooking heat/smoke vents outside.
4. **Room 2 & Room 3 doors at their south ends**, away from the kitchen.
5. **Verandah outside the 48'** — interior stays a clean 1,200 sqft.
6. **Every habitable room** has windows on two external walls (cross-ventilation).
7. **Future-proofed:** columns, footings, stair all sized for a 2nd floor.

## 5. Critical Warnings ⚠️

1. **Soil test before footings** — sizes assume SBC 75 kN/m².
2. **Cantilever verandah top steel** is structural — never cut it.
3. **Septic ≥ 50' from tube well**, downstream.
4. **Column starter bars** must be protected for Phase-2.
5. Confirm all rates locally before ordering (prices move).
