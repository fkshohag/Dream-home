# 🏡 Dream Home — 48' × 25' Single-Story House (1,200 sqft)

A modern tropical, future-expandable home for a Bangladeshi village plot. Main entry + verandah on the **south**, secondary door on the **west**, central-dining pinwheel layout, kitchen vented through the north wall, and a structure already engineered for a **2nd floor**.

> Full document index → **[`house-design.md`](house-design.md)**

---

## Quick Stats

| | |
|---|---|
| 🏠 Footprint | 48' (N–S) × 25' (E–W) = **1,200 sqft** + 150 sqft verandah |
| 🛏️ Bedrooms | 3 (2 masters w/ attached bath + 1 with common bath) |
| 🚪 Entries | Main (south, verandah) + secondary (west door) |
| 🏗️ Structure | RCC frame · 15 columns · isolated footings · G+1 ready |
| 💰 Core cost | **≈ 50–52 lakh** (~4,280 BDT/sqft) |
| 💰 Core + extended | **≈ 84 lakh** (boundary, solar, premium kitchen, etc.) |
| 📐 Code | BNBC 2020, Seismic Zone-2 |

## Critical reminders ⚠️

1. **Soil test before footings.**
2. **Don't cut the verandah cantilever top steel.**
3. **Septic ≥ 50' from the tube well.**
4. **Protect column starter bars** for the future 2nd floor.
5. **Verify all material rates locally** before ordering.
